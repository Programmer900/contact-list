# contact-list
Project on vuetify

## Установка зависимостей(обязательно)
```
npm install
```

### Компиляция, быстрый пересбор для разработки(для проверки на локальном)
```
npm run serve
```

### Компиляция и минификация для Продакшн
```
npm run build
```

### Линтер и фикс файлов
```
npm run lint
```
